Напрямую менять что-либо из классов пакета entity запрещено, после изменений в changelog файлах запустите приложение.
Структура базы автоматически обновится посредством liquibase.
Остановите приложение.
Следующий шаг - сгенерировать JPA Entity классы с помощью плагина hibernate3-maven-plugin:

mvn hibernate3:hbm2java

JPA Entity классы появется в пакете entity.
При необходимости поправьте другие связанные с ними классы.
